# weewx-clearskies-design-tokens

Phase 1 placeholder — populated during Phase 6+ of the Clear Skies project.

Tailwind config + named design variables (palette, spacing, typography) for the [Clear Skies](https://github.com/inguy24/weewx-clearskies-stack) project. Eventually published as a standalone npm package so others can build dashboards against the same visual language.

For Phase 1–5, design tokens live inside [weewx-clearskies-dashboard](https://github.com/inguy24/weewx-clearskies-dashboard); this repo is a name reservation per [ADR-001](https://github.com/inguy24/weewx-belchertown/blob/master/docs/decisions/ADR-001-component-breakdown.md).
